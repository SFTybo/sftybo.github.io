I am currently 17 years old in my 3rd year of high school. My interest in software development began when I was around 10 years old and I tried using game engines like Unity and the ROBLOX platform to create games. Starting out wasn't easy but soon I was picking up everything from game and web design, to 3d modelling. For the past several years I've been learning more and more about programming. I am responsible for the development of the Code Ninjas ROBLOX game that's been played during the Halloween event and the Thanksgiving event as well as upcoming birthdays and other events. During the summer camps, I helped to manage the Code Ninjas Minecraft servers not only for the ninjas' free play time, but as well as putting together all of their TinkerCAD builds into one Minecraft world to play in. I've been working at Code Ninjas for a year now and it's been great so far. My plan for the future is to become a game developer.
